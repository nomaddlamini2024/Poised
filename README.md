# Poised
Poised repository is about creating a project management system for a small structural engineering firm called “Poised” – i.e., it's about creating a program to keep track of projects on which they work.

Project's Title
Poised Project Management System (PMS)
Poised Project Management System (PMS) is a program put together for a small structural engineering firm, “Poised”. Poised will use the system to keep track of their projects.

Project Description
Poised PMS is about putting together a Java program to create a project management system.
The system will be used to keep track of projects and to manage projects that Poised has taken on or are already working on or are in the pipeline for work to be initiated.
What the application does
The PoisedPMS class in the system provides a menu-driven program to manage projects and associated personnel in a database. It supports adding, updating, deleting, finalizing projects and viewing specific project details.

Why the project is useful
Users will be able to capture information about new projects 
They will also be able to add new projects to the database.
The application will enable users to update information about existing projects.
Users will be able to delete data about projects and people associated with them.
The system provides capability to the user to finalise existing projects - by marking finalised projects as “finalised” as well adding completion dates
The application will enable users to conduct a search on projects that still need to be completed from the database.
Users will be able to find and select a project by entering either the project number or project name.

The following pertinent information about projects and personal will be stored in the system’s database, making it easy to keep track of projects:
  . Project number
  . Project name.
  . The type of building is being designed
  . The physical address for the project
  . The ERF number
  . The total fee being charged for the project
  . The total amount paid to date.
  . Deadline for the project.
  . The name, telephone number, email address and physical address of the architect for the project.
  . The name, telephone number, email address and physical address of the contractor for the 
    project.
  . The name, telephone number, email address and physical address of the customer for the project.

How users can get started with the project
To run the Poised Project Management System Java program, the User will need to do the following:

Step 1: Install Java Development Kit (JDK) on their computer. 
The JDK can be downloaded from the Oracle website or any other reliable source.
The User needs to download the JDK for their operating system – e.g., if the use Windows, they would need to download the JDK version for Windows

Step 2: Copy and paste the PoisedPMS.java file in a code editor of their choice
The file must be saved with a .java extension and file type as ‘All files’ in the desired location in User’s computer

Step 3: Open the command prompt 
If the User is using Windows, they can open a Windows command prompt
C:\Users\joesoap>

Step 4: Navigate to the directory where the User saved their Java code using the “cd” command.
For example, for a file saved on a C drive
C:\Users\joesoap>cd  desktop
C:\Users\joesoap\Desktop>cd  task8
C:\Users\joesoap\Desktop\task8>

Step 5: Compile the PoisedPMS.java file code by typing “javac [filename].java” – i.e., PoisedPMS.java       in the command prompt/terminal. 
This will create a .class file in the same directory.

Step 6: Running the PoisedPMS Java program by typing “java [filename]” – i.e., PoisedPMS.java in the command prompt/terminal. The program will execute and produce the output.
